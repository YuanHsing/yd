###  :fa-arrows: 【导入方案】

- ［方案一］
- 导入书源前，请先在阅读APP内校验一遍旧书源，并删除失效书源；再将本站书源进行覆盖导入即可。

- ［方案二］
- 可直接删除阅读APP里旧书源，从新导入本站书源。(推荐使用此方案，方便又快捷)


###  :fa-film: 【导入教程】

- 复制【书源链接】，打开阅读，选择【书源管理】，点击右上角三个点，选择【网络导入】，粘贴链接，确定即可。
- 将本站【书源二维码】保存至本地，打开阅读，选择【书源管理】，点击右上角三个点，选择【二维码导入】，等待导入成功即可。
